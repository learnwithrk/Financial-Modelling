1.3 Adding Totals and Growth %

  • [cite_start]Total sales: =SUM(B2:G2) [cite: 21]
  • [cite_start]Average annual sales: =AVERAGE(B2:G2) [cite: 22]
  • [cite_start]Growth %: =(G2-B2)/B2 (format cell as Percentage) [cite: 23]

1.4 Instructions

[cite_start]Creating a Line Chart [cite: 24] 1. [cite_start]Select A1:G5 (headers + all data) [cite: 25] 2. [cite_start]Go to Insert → Charts → Line → Line with Markers [cite: 26] 3. [cite_start]Right-click chart → Add Chart Title: ‘Regional Sales Trend 2015–2020’ [cite: 27] 4. [cite_start]Click ‘+’ icon on chart → Check Axis Titles → Type: Horizontal=‘Year’, Vertical=‘Sales (Rs)’ [cite: 28] 5. [cite_start]Right-click Legend → Format Legend → Bottom [cite: 29]

[cite_start]Sorting the Data [cite: 30] 1. [cite_start]Click inside the data → Data tab → Sort [cite: 31] 2. [cite_start]Primary sort: Year (Oldest to Newest) → Add Level → Secondary: Revenue (Largest to Smallest) [cite: 32]

[cite_start]Advanced Filter [cite: 33] 1. [cite_start]Copy your column headers to a blank area (e.g., Row 30) [cite: 34] 2. [cite_start]Type filter criteria below those headers (e.g., Year=2019, Region=North) [cite: 35] 3. [cite_start]Data → Advanced → Filter → Copy to another location → Set List Range, Criteria Range, Copy To [cite: 36]

[cite_start]📌 Note: The Criteria range must have EXACT column headers matching the data table[cite: 37]. [cite_start]Use commas between criteria in same row for AND conditions; different rows for OR[cite: 38].

[cite_start]✅ Result: North region: 2015–2020 CAGR ≈ 9.2% (highest)[cite: 39]. [cite_start]All regions show consistent upward trend[cite: 39].
