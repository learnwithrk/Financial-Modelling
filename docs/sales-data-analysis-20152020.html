FINANCIAL MODELLING

Chapter 1 Sales Data Analysis (2015–2020)

1.1 Learning Outcomes

After completing this experiment, students will be able to: - Enter and validate data in Excel - Apply formulas (SUM, AVERAGE) - Create and format charts (Line/Column) - Export results to Word - Perform Sorting and Advanced Filtering

1.2 Dataset (As per Excel File)

Year Product Region Sales
2015 A South 50000
2016 A South 60000
2017 A South 65000
2018 A South 70000
2019 A South 80000
2020 A South 90000

1.2.1 📁 Download Excel File

Download Sales Data Excel

1.3 Step 1: Data Entry & Formulas (Based on Excel File)

1.3.1 Procedure

  1. Open Microsoft Excel → New Workbook
  2. Enter the dataset in columns A–D
  3. In column D, enter Sales values
  4. Leave column E for calculations

1.3.2 Formulas (Already Embedded in Excel File)

Cell Description Formula
E9 Total Sales =SUM(D2:D7)
E10 Average Sales =AVERAGE(D2:D7)

1.3.3 Labels in Excel

  • D9 → “Total”
  • D10 → “Average”

👉 This structure is already implemented in the provided Excel file

1.4 Step 2: Chart Creation

1.4.1 Procedure

  1. Select range A1:D7
  2. Insert → Charts
  3. Choose:
    • Line Chart (Trend Analysis)
    • Column Chart (Comparison)

1.4.2 Diagram: Chart Creation Flow

flowchart LR
A[Select Data] --> B[Insert Tab]
B --> C[Choose Chart Type]
C --> D[Line Chart]
C --> E[Column Chart]

1.4.3 Expected Output

  • Sales increase steadily from 2015 to 2020

1.5 Step 3: Chart Formatting

1.5.1 Procedure

  • Add Chart Title: Sales Trend (2015–2020)
  • Add Axis Titles (Year, Sales)
  • Add Legend
  • Apply styles

1.5.2 Diagram: Chart Elements

graph TD

A --> D[Y Axis]

1.6 Step 4: Export to Word

1.6.1 Procedure

  1. Copy Excel table → Paste in Word
  2. Copy chart → Paste in Word

1.6.2 Interpretation

The sales data shows a consistent upward trend from 2015 to 2020. The increase in total and average sales indicates steady business growth.

1.7 Step 5: Sorting

1.7.1 Procedure

  1. Select dataset (A1:D7)
  2. Data → Sort
  3. Apply:
    • Year → Ascending
    • Sales → Descending

1.7.2 Diagram

flowchart TD
A[Dataset] --> B[Sort]
B --> C[Ascending]
B --> D[Descending]

This experiment demonstrates how Excel tools can be used effectively for structured data analysis, visualization, and decision-making using real formulas embedded in the worksheet.
