📖 HOW TO USE THIS MANUAL

This manual provides complete step-by-step instructions for all 20 financial Excel exercises. Use it alongside the Excel workbook Financial_Excel_Exercises_Complete.xlsx.

Each exercise section contains: * Objective — the learning goal * Data table — what to enter in Excel * Step-by-step instructions with exact Excel menu paths * Key formulas with full explanation * Expected results and interpretation

📌 Note: Open the Excel workbook first. Each sheet tab corresponds to one exercise. Yellow cells = user input. Green cells = calculated results.
